## By Psychological Principle

| Principle | Location | Domains | Key Insight |
|-----------|----------|---------|------------|
| **Autonomy** | Psych-Foundations | All | Freedom to choose sustains engagement |
| **Competence** | Psych-Foundations + Feedback | All | Visible progress motivates |
| **Relatedness** | Psych-Foundations | All | Belonging drives loyalty |
| **Scarcity** | Behavioral-Triggers | Games, Marketing | Limited availability increases value; must be honest |
| **Urgency** | Behavioral-Triggers | Games, Marketing | Time pressure activates action; overuse habituates |
| **Social Proof** | Behavioral-Triggers | All | Consensus signals correctness; must be authentic |
| **Authority** | Behavioral-Triggers | All | Credible experts are trusted; fake credentials destroy trust |
| **Reciprocity** | Behavioral-Triggers | All | Giving value first creates obligation; must be genuine |
| **Commitment** | Behavioral-Triggers | All | Small commitments escalate to larger |
| **Liking** | Behavioral-Triggers | All | We comply with those we like; similarity increases liking |
| **Novelty** | Behavioral-Triggers | All | New stimuli activate dopamine; optimal novelty level matters |
| **Loss Aversion** | Prospect Theory | All | Losses 2x more powerful than gains; use carefully |
| **Framing** | Framing-Effects | All | Same fact, different frame = different response |
| **Feedback** | Feedback-Loops | All | Immediate, earned, specific feedback sustains motivation |
| **Habit** | Motivation-Drivers | All | Cue + routine + reward creates automatic behavior |
| **Flow** | Motivation-Drivers + Research | Games | Optimal challenge level maintains engagement |
