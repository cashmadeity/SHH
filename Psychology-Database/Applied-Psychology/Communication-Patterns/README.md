# Communication Patterns Knowledge Base

## Overview

This knowledge base documents evidence-based communication psychology applied to three priority domains:
- **Video Games** - Player engagement, community dynamics, feedback loops
- **Marketing** - Persuasion, behavioral triggers, audience segmentation
- **Therapy** - Therapeutic alliance, behavior change, trauma-informed messaging

## Key Concepts Covered

- **Psychological Foundations**: Self-Determination Theory, Social Proof, Authority Bias, Reciprocity
- **Message Design**: Framing, Priming, Scarcity, Urgency, Loss Aversion
- **Engagement**: Feedback systems, motivation, habit formation, community psychology
- **Domain-Specific Applications**: Implementation patterns for games, marketing, therapy
