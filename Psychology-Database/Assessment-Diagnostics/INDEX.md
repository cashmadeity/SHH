## Key Takeaways (One-Minute Summary)

```
ASSESSMENT ETHICS IN A NUTSHELL:

1. Use VALID assessments (peer-reviewed evidence for YOUR population)
2. Stay in your COMPETENCE lane (trained, supervised, know your limits)
3. DETECT & MITIGATE BIAS (fairness audits, diverse data, demographic testing)
4. Get REAL INFORMED CONSENT (plain language, comprehension check, voluntariness)
5. Adapt for CULTURE (not just translation; validation required)
6. SECURE DATA HARD (encryption, access control, audit logs, breach plan)
7. AUDIT before launch & MONITOR after (5-level audit framework)

Miss ANY of these → Ethical violation → Legal/professional liability
```

## Getting Started: Choose Your Path

### Path A: "I need to launch an assessment app in 8 weeks"
1. READ: [01](01-Ethical-Assessment-Principles.md) (30 min)
2. RUN: Level 1 audit ([07](07-Assessment-Ethics-Audit-Framework.md), Section 1) (4 hours)
3. PLAN: Levels 2-4 (1 week timeline)
4. DO: Level 2 review ([07](07-Assessment-Ethics-Audit-Framework.md), Section 2) (3 hours, hire external)
5. DO: User testing ([07](07-Assessment-Ethics-Audit-Framework.md), Section 4) (4 weeks, 50+ users)
6. LAUNCH (after addressing feedback)

**Total time:** 2 days reading + 1-2 weeks review + 4 weeks testing = ~6 weeks total

### Path B: "I want to build a culturally-adapted assessment"
1. READ: [05](05-Cultural-Adaptation-of-Assessments.md) (30 min)
2. ASSESS: Original assessment (validity? population match?)
3. PLAN: Adaptation strategy (translation + validation + norms?)
4. DO: Cognitive testing with target users ([05](05-Cultural-Adaptation-of-Assessments.md), Step 2)
5. DO: Validation study ([05](05-Cultural-Adaptation-of-Assessments.md), Step 3) (2-3 months)
6. RUN: Level 1 audit with new culture data ([07](07-Assessment-Ethics-Audit-Framework.md))
7. LAUNCH with local norms & interpretation guide

**Total time:** 3-4 months (validation is time-intensive)

### Path C: "I need to audit an existing assessment"
1. READ: [07](07-Assessment-Ethics-Audit-Framework.md) (25 min)
2. RUN: Level 1 self-audit (4-6 hours)
3. HIRE: External expert for Level 2-3 (1-2 weeks)
4. PLAN: Fixes for gaps (timeline?)
5. SET UP: Level 5 monitoring ([07](07-Assessment-Ethics-Audit-Framework.md), Section 5)
6. REPORT: Annual audit findings to leadership

**Total time:** 2 days of your time + 1-2 weeks external review
