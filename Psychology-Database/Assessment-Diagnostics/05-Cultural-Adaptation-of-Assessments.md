# Cultural Adaptation of Assessments

**Source:** APA Standards 9.01, 9.02, 9.06; ITC Guidelines for Adapting Tests (Hambleton et al., 2005); DSM-5-TR Cultural Formulation  
**Status:** Critical for equitable assessment across cultures  
**Scope:** Validating assessments across cultural groups, adapting for cultural context

---

## The Problem: One-Size-Fits-All Assessment

**Myth:** If a test is validated, it works for everyone.

**Reality:** Tests are validated on specific populations. Using on different culture = untested, unreliable.

### Examples of Cultural Misfit

| Test | Designed For | Problem When Used Across Cultures |
|------|---|---|
| **PHQ-9 (Depression)** | Western, individualistic | Collectivist cultures: Depression expressed through family/social function, not individual mood |
| **ADHD Screening** | Industrialized Western | Different cultures: Activity levels, attention norms vary (sitting still not valued equally) |
| **IQ Testing** | White middle-class Americans | Non-Western cultures: Different learning styles, problem-solving approaches not captured |
| **PTSD Criteria** | Western trauma response | Non-Western cultures: Spiritual/somatic expression of trauma (not just anxiety/flashbacks) |
| **Anxiety Screening** | English-speaking Western | Translated to Spanish: Some words don't translate; meaning changes |

## Levels of Cultural Adaptation

### Level 1: Translation Only (Minimum, Often Insufficient)

**What:** Translate test into another language (directly, word-by-word)

**Process:**
- Hire translator
- Translate items
- Back-translate to check
- Done

**Problem:**
- Language ≠ culture
- Idioms don't translate
- Word meanings change (e.g., "sad" might not exist in some cultures with different emotion taxonomy)
- Norms still based on original culture
- No validation in new culture

**Example of mistranslation:**
- English: "I feel blue" (sad)
- Literal Spanish: "Me siento azul" (makes no sense; azure/blue is not an emotion concept)
- Correct: "Me siento triste" (sad)

### Level 2: Linguistic Adaptation (Better)

**What:** Adapt language to cultural meaning while preserving construct

**Process:**
- Hire bilingual psychologist (not just translator)
- Adapt items for cultural meaning (not just translate)
- Cognitive testing (ask users if items make sense)
- Back-translate & compare
- Validate semantic equivalence

**Example:**
- Original: "I feel guilt about things I've done"
- Chinese adaptation: "I feel shame about burdening my family" (guilt → shame/obligation is the emotion in that culture)
- Validation: Do both items predict depression equally in both cultures?

### Level 3: Contextual Adaptation (Better Yet)

**What:** Adapt items AND interpretation for cultural context

**Process:**
- Linguistic adaptation (from Level 2)
- Plus: Add contextual questions
- Understand cultural meaning of symptoms
- Different cutoff scores if appropriate
- Culturally-informed interpretation

**Example:**
- Depression screening for grief-stricken person
- Standard adaptation asks: "Have you experienced loss recently?"
- Better adaptation: "Loss can be expected sadness, not always depression. Did your sadness start with a loss?"
- Interpretation: Same score might be normal grief in collectivist culture (family obligation), not depression

### Level 4: Full Test Adaptation (Best, Most Expensive)

**What:** Modify test content, norms, interpretation for culture

**Process:**
- Linguistic adaptation
- Contextual adaptation
- Complete re-validation on target culture
- New norms for that culture
- Cultural formulation interview (DSM-5-TR approach)
- Clinician training in cultural interpretation

**Example: Depression Screening for South Asian Culture**
- Items modified to include somatic symptoms (cultural expression of depression)
- Add items about family obligation/shame (cultural meaning)
- Norms re-computed on South Asian population (cutoff scores differ)
- Interpretation training: Same score doesn't mean same level of depression across cultures

## ITC Guidelines: The Gold Standard for Adaptation

**International Test Commission (ITC)** provides best-practice adaptation guidelines:

### Guideline 1: Context & Rationale

```
Before adapting, document:
[ ] Why is adaptation needed? (new language? new culture? both?)
[ ] Who will use adapted version? (clinicians? researchers? non-professionals?)
[ ] What's the purpose? (clinical decision? research? screening?)
[ ] What population? (age, education, health status, etc.)
[ ] Is adaptation legally required? (e.g., bilingual school systems)

This defines scope of adaptation work.
```

### Guideline 2: Development & Adaptation

```
Assembly phase:
[ ] Review source test (what does it measure? how validated?)
[ ] Identify construct (what you're really measuring)
[ ] Identify items that might not work across cultures
[ ] Gather expert input (bilingual clinicians, cultural consultants)

Design phase:
[ ] Decide: Translate? Adapt items? Change content? New norms?
[ ] Hire bilingual/bicultural adapters (not just translators)
[ ] Create initial adapted version
[ ] Cognitive testing (does it make sense to target users?)

Refinement phase:
[ ] Back-translation (compare source & back-translated)
[ ] Item review (is semantic equivalence achieved?)
[ ] Final version ready for validation
```

### Guideline 3: Validation

```
ESSENTIAL: Validate on new culture (not just translation!)

Step 1: Reliability
- Does adapted test have good internal consistency?
- Test-retest reliability (scores stable over time)?
- Split-half reliability?

Step 2: Validity
- Criterion validity: Does it predict what it should?
  (Does adapted depression screening predict clinical diagnosis in target culture?)
- Construct validity: Does it measure same thing as original?
  (Does adapted version measure depression similarly?)
- Discriminant validity: Doesn't measure other constructs?

Step 3: Norms
- Re-norm on target population (don't use original culture norms!)
- Percentiles for target culture
- Understand if cutoff scores need adjustment

Step 4: Differential Item Functioning (DIF)
- Do items work equally for all subgroups in target culture?
- (e.g., Men & women? Urban & rural? Different education levels?)
```

### Guideline 4: Documentation & Training

```
Document everything:
[ ] Adaptation decisions & rationale
[ ] Validation results
[ ] Known limitations
[ ] Recommended cutoff scores for target culture
[ ] How to interpret results (what scores mean in context)

Train adapters & users:
[ ] Why items were changed
[ ] What cultural meaning they capture
[ ] How to interpret in context
[ ] When to refer (different triggers in different cultures)
```

## DSM-5-TR Cultural Formulation Approach

**Best practice for assessment across cultures: Cultural Formulation Interview (CFI)**

Instead of just translating diagnostic criteria, add cultural context:

### CFI Core Question Categories

```
1. CULTURAL IDENTITY
   - How does user identify culturally/ethnically?
   - What's important about their identity?
   - How does this affect their mental health view?

2. CULTURAL EXPLANATIONS
   - What do they think is causing their problem?
   - Is it a diagnosis? A spiritual/moral issue? A curse?
   - What do their family/community think?
   
   Example: Depression might be seen as:
   - Western: Clinical disorder (need medication)
   - Collectivist: Family shame (need to restore honor)
   - Spiritual: Evil eye (need spiritual intervention)
   - Somatic: Physical illness (need medical exam)

3. CULTURAL FEATURES AFFECTING DIAGNOSIS
   - Are symptoms culturally normal or unusual?
   - Would diagnosis be helpful or stigmatizing?
   - How would diagnosis be received?
   
   Example: Hearing voices
   - Psychosis in Western culture
   - Spiritual communication in some cultures (normal/valued)
   - Diagnosis must account for cultural context

4. CULTURAL FACTORS AFFECTING TREATMENT
   - What treatment does user prefer?
   - Are they culturally acceptable?
   - What's the role of family/community?
   - Are there cultural barriers to Western treatment?
   
   Example: Therapy
   - Western: Individual therapy (talking about feelings)
   - Collectivist: Family therapy (improve relationships)
   - Some cultures: Preferred not to discuss mental illness with outsider

5. OVERALL CULTURAL ASSESSMENT
   - How do cultural factors influence diagnosis?
   - Do criteria apply equally in this cultural context?
   - Is diagnosis necessary? Helpful? Harmful?
```

## Common Cultural Misadaptations (What NOT to Do)

### Misadaptation #1: Direct Translation

**Example:**
- English assessment: "I feel worthless"
- Direct Spanish translation: "Me siento sin valor"
- Problem: Doesn't capture cultural shame/honor meaning

**Fix:**
- Work with bilingual clinician to find culturally-equivalent concept
- Cognitive testing with target users
- Validate that adapted item predicts same outcome

### Misadaptation #2: Ignoring Somatic Expression

**Example:**
- Depression screening asks only about mood
- Ignores that in many cultures, depression = physical symptoms (body aches, fatigue, headaches)

**Fix:**
- Add somatic items: "Do you have physical pain or heaviness?"
- Validate that these items predict depression in target culture

### Misadaptation #3: Wrong Norms

**Example:**
- Anxiety screening validated on 1,000 Americans
- Used with Middle Eastern refugees
- Same cutoff score (says 20% are anxious)
- But refugees have real trauma (actually 60% have PTSD symptoms)
- Tool missing half the cases because norms are wrong

**Fix:**
- Re-norm on target population
- Adjust cutoff scores based on new culture
- Validate that new cutoff scores work

### Misadaptation #4: Ignoring Cultural Context in Interpretation

**Example:**
- Youth in collectivist culture scores high on "I don't fit in"
- Interpreter thinks: Social anxiety disorder
- Reality: Normal adolescent adjustment to family expectations
- Wrong diagnosis

**Fix:**
- Use Cultural Formulation Interview
- Ask: "In your culture, is it normal for teens to feel this way?"
- Interpret scores relative to cultural context

### Misadaptation #5: Forcing Western Diagnosis on Non-Western Culture

**Example:**
- Assessment diagnoses "Major Depressive Disorder" in Kenyan client
- DSM criteria met
- But client's community sees it as "spiritual imbalance"
- Diagnosis doesn't match cultural meaning
- Client rejects treatment

**Fix:**
- Offer both: DSM diagnosis + cultural formulation
- Say: "In Western psychology, this is depression. In your community, what would you call this?"
- Respect cultural understanding while offering Western treatment option

## Building Cultural Validity into Assessment

### Step 1: Identify Cultural Dimensions

Before adaptation, understand target culture:

```
CULTURAL DIMENSION ANALYSIS

Individualism-Collectivism:
- Individualistic: Personal goals, individual identity valued
- Collectivist: Family/community goals, relational identity valued
- Assessment implication: How are symptoms expressed?
  (Individual mood vs. family function vs. social role)

Power Distance:
- Low: Egalitarian, question authority
- High: Hierarchical, respect authority
- Assessment implication: How does power in assessment affect honesty?
  (Will minority user tell authority figure truth? Why not?)

Uncertainty Avoidance:
- Low: Tolerates ambiguity, risk-taking
- High: Prefers rules, certainty, structure
- Assessment implication: How much detail needed?
  (Vague items frustrate high-UA cultures; need clarity)

Masculinity-Femininity:
- Masculine: Competition, achievement, assertiveness
- Feminine: Cooperation, relationships, wellbeing
- Assessment implication: How are emotions expressed?
  (Assertiveness vs. harmony; independence vs. interdependence)

Time Orientation:
- Past: Honor tradition, ancestors, history
- Present: Live in moment, immediate concerns
- Future: Plan ahead, delayed gratification
- Assessment implication: Temporal questions relevant?
  (Past trauma focus might miss present concerns in present-oriented culture)
```

### Step 2: Cognitive Testing (Does It Make Sense?)

Before validation, pilot with target users:

```
COGNITIVE TESTING PROTOCOL

Interview 8-12 target users:
1. Read item aloud
2. Ask: "What does this question mean to you?"
3. Ask: "Does this make sense in your culture?"
4. Ask: "Would you answer it the same way as [comparison item]?"
5. Ask: "Is there a better way to ask this?"

Document:
- Comprehension issues
- Cultural mismatches
- Suggested modifications

Revise items based on feedback
```

### Step 3: Validation Study Design

```
VALIDATION STUDY

Sample:
- n = 200+ from target culture (large enough to get stable statistics)
- Representative of target population (age, gender, education, etc.)
- Fluent in target language

Measures:
- Adapted assessment
- Gold-standard criterion (clinical diagnosis, expert rating)
- Original assessment (if possible, to compare)

Analyses:
- Reliability (internal consistency, test-retest)
- Validity (sensitivity, specificity, accuracy vs. gold standard)
- Differential item functioning (do items work for all subgroups?)
- Compare to original assessment (is it equivalent?)

Report:
- Validation results (psychometric properties)
- Recommendations for use
- Known limitations
- Recommended cutoff scores for target culture
```

### Step 4: Interpreter/Clinician Training

```
TRAINING FOR THOSE USING ADAPTED ASSESSMENT

What they need to know:
- What changed from original? Why?
- What is this assessment measuring in target culture?
- What do scores mean? (Norms for target population)
- When does this apply? When not? (Limitations)
- How to interpret in cultural context?
  (What's normal for this culture vs. truly pathological?)
- How to integrate cultural formulation?
- When to refer?

Training modality:
- Workshop (2-4 hours minimum)
- Or online course (if asynchronous)
- Case examples in target culture
- Practice interpretation
- Quiz/certification (ensure competence)
```

## Cultural Adaptation Checklist: Pre-Launch

```
ASSESSMENT ADAPTATION
[ ] Identified need for adaptation (not just translation)
[ ] Cultural dimensions analyzed
[ ] Bilingual/bicultural team assembled
[ ] Items reviewed for cultural fit

LINGUISTIC ADAPTATION
[ ] Bilingual clinician led adaptation (not just translator)
[ ] Cognitive testing with 8+ target users
[ ] Back-translation confirms semantic equivalence
[ ] Items revised based on feedback

VALIDATION
[ ] Validation study conducted on target culture
[ ] Sample size adequate (200+)
[ ] Reliability data: Internal consistency, test-retest
[ ] Validity data: Sensitivity, specificity, vs. gold standard
[ ] DIF analysis: Items work equally for subgroups
[ ] Comparison to original: Equivalence confirmed

NORMS & CUTOFFS
[ ] New norms computed for target culture
[ ] Cutoff scores adjusted (if different from original)
[ ] Percentile equivalents calculated
[ ] Documentation: What scores mean in target culture

INTERPRETATION GUIDE
[ ] Plain-language interpretation guide created
[ ] Cultural formulation approach documented
[ ] Examples given (interpretation in context)
[ ] Limitations stated explicitly
[ ] When to refer documented

TRAINING
[ ] Training curriculum developed
[ ] Clinicians trained on adapted assessment
[ ] Cultural competence emphasized
[ ] Certification/competence verification done

ONGOING
[ ] Monitoring for cultural fit (user feedback)
[ ] Annual review: Does adaptation still apply?
[ ] New demographic groups tested (within-culture variation)
[ ] Continuous improvement based on use
```

## Summary: When Assessment Crosses Cultures

```
✅ GOOD: Adapted Assessment
- Validated on target culture
- Items reflect cultural meaning
- Norms from target population
- Interpretation culturally-informed
- Clinician trained in cultural use

❌ BAD: Direct Translation
- No validation on target culture
- Items translated, not adapted
- Original norms used (wrong)
- Interpretation same as original (may not apply)
- Clinician not trained in cultural differences

❌ WORSE: Untranslated Assessment Used in Second Language
- Assessment in English
- User fluent in Spanish
- Language barrier makes assessment unreliable
- Diagnostic error likely
```
